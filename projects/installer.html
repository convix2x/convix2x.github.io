installer

Easily install Windows 8.x apps.

Installer is a simple application to install .appx files on Windows 8. In future updates, the Sources menu will also be released, where you can add repositories, and install apps over the network from these repositories.

You can find source tarballs for Installer on the github repo convix2x/installer.
